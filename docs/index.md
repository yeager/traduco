# LinguaEdit Documentation

**LinguaEdit** is a modern, cross-platform translation file editor built with Qt 6 (PySide6). It supports a wide range of localisation file formats and provides professional-grade tools for translators.

## Overview

LinguaEdit combines the familiar interface of editors like POedit and Qt Linguist with modern features such as AI-powered pre-translation, translation memory, glossary enforcement, and integration with popular translation platforms.

### Key Capabilities

- **Multi-format editing** — PO/POT, Qt TS, JSON, XLIFF 1.2/2.0, Android XML, Flutter ARB, PHP arrays, YAML
- **Tabbed editing** — work on multiple files simultaneously
- **Inline linting** — real-time quality checks with format specifier validation
- **Pre-translation** — powered by Lingva, MyMemory, OpenAI, Anthropic, DeepL, and more
- **Translation memory** — fuzzy matching from previously translated content
- **Glossary manager** — enforce consistent terminology across your project
- **Platform integration** — Transifex, Weblate, Crowdin, and GitHub PR workflows
- **Git integration** — view status, diffs, and commit directly from the editor
